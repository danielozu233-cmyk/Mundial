Regla de Oro: Tienes hasta exactamente 1 hora antes del inicio programado del partido para ingresar o modificar tu pronóstico. Pasado ese tiempo, el partido se bloquea automáticamente y ya no podrás guardar cambios para ese encuentro.

El sistema de puntuación está diseñado para premiar la exactitud:

  • 4 PTS Acertar el marcador exacto del partido.
  • 2 PTS Acertar la tendencia (ganador o empate), pero no el marcador exacto.
  • 0 PTS Fallar completamente el resultado.

Criterio de Desempate

Si dos o más participantes terminan con la misma cantidad de puntos totales, el criterio de desempate será el siguiente: Gana quien tenga más partidos acertados con puntuación perfecta (4 puntos).
